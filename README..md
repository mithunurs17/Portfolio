# Portfolio Project

This repository contains a personal portfolio application built with React, Vite, Tailwind CSS, and Express.

## Project structure

- `client/` - front-end application code
- `server/` - backend server and routes
- `shared/` - shared schema definitions
- `src/` - client source files, including components and pages

## Technologies

- React 18
- Vite
- TypeScript
- Tailwind CSS
- Express
- Drizzle ORM
- Passport authentication

## Scripts

- `npm run dev` - start the development server
- `npm run build` - build the client and bundle the server
- `npm start` - start the production server from `dist`
- `npm run check` - run TypeScript type checking
- `npm run db:push` - push database schema changes with Drizzle

## Getting started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start development:
   ```bash
   npm run dev
   ```

## Notes

Adjust environment variables and database configuration as needed for your local setup.
